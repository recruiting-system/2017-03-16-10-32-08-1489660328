# ruby-Fizz-Buzz

#### 安装test-unit
```
$ sudo gem install test-unit
```

#### 编写一个程序将１到n的数字以字符串输出，要求如下：

1. 当数字为3的倍数时输出“Fizz”
2. 当数字为5的倍数时输出“Buzz”
3. 当数字既是3的倍数，又是5的倍数时输出“FizzBuzz”

例如:
```
n = 15,
```
输出为:
```
[
    "1",
    "2",
    "Fizz",
    "4",
    "Buzz",
    "Fizz",
    "7",
    "8",
    "Fizz",
    "Buzz",
    "11",
    "Fizz",
    "13",
    "14",
    "FizzBuzz"
]
```
